# UPSC Shorts

A current-affairs micro-learning app for UPSC aspirants, with a built-in **Maps Arcade** geography quiz.

## Run
Open `index.html` in a browser (or host the folder on GitHub Pages).

## Files
- `index.html` — the UPSC Shorts app
- `index-3.html` + `support.js` — Maps Arcade (world & India geography quizzes)
- `data/` — India rivers & national-parks map data

## Daily content
Generate a `shorts-YYYY-MM-DD.json` from a current-affairs PDF, then in-app: **+ → Import JSON**.
